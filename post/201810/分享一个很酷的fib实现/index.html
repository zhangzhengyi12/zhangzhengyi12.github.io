分享一个Fib数列实现 [reduce]

Posted by Yinode on Tuesday, October 16, 2018

TOC

const fib = n =>
  Array(n)
    .fill(0)
    .reduce(pre => [pre[1], pre[1] + pre[0]], [0, 1])[0]

def fib(n): return reduce(lambda x, n: [x[1], x[0]+x[1]], range(n), [0, 1])[0]
