前提

康德提出了物自体不可知的理论，当然也会造成一个二元对立的难题，所以后面的很多后康德哲学家的首要意图就是想消解这种二元对立，用一个统一的理论来组织起主客二分，在这里谢林也是其中一员，他的方法说白就是证明，主客本来就是同一个东西，他需要后撤到一个绝对可靠的基点之中，然后从这个点推导出所有东西的构成，既然所有东西都是从一个点诞生，或者说都是同一种力量，那么也就自然解决了这个二元对立的问题。

名词解释

  1. 思维的同一性 自我的二重性，也就是创造和被创造的同一(个人的理解)
  2. 本源性的知识 最本质，最基本，最原子化的知识，让知识得以可能的知识
  3. 综合命题 主词和宾语超越思维的同一性(也就是不受思维的制约)，并通过某种外在思维的东西进行中介的命题 这种命题是构成知识的基础。
  4. 分析命题 被限制在思维之内的命题，满足思维同一性的命题
  5. 无条件的知识 不和任何无条件的知识有关，只和主观的东西有关
  6. 有条件的知识 和无条件的知识有关

导论

什么是先验哲学

我们把单纯客观的东西，称之为自然，单纯主观的东西，称为自我

  1. 把客观的东西当作第一位的，让主观依附于客观，这是自然科学，自然科学趋向于把自然理智化，这是自然哲学
  2. 把主观的东西当作第一位的，让客观依附于主观，这是先验哲学，让客观的东西从主观的东西里面产生出来

但是对于哲学来说，解释这种自然从理智中诞生的活动的先验哲学就是哲学的一门必不可少的科学!

推论

先验哲学和自然哲学两者互相对立，但是又在向对方的方向进行发展，相互需求和补充。现在，我们需要对进行先验哲学的思考进行一些基础准备。

首先，既然先验哲学想要说明一切实在的根本根据，那么必须从普遍的怀疑开始，而彻底的怀疑必须针对根本成见。在一切成见之中有一个根本性的成见，他就是我之外有物，那么为什么我们无法确定这种成见的必然性，但却直接接受了这种命题呢？我们暂且推断我们存在一个确定的命题，并且这个命题与这个成见根本上是同一种东西!说明这种同一性，是先验哲学必须做的事。

从理性的角度上说，除了我在这个命题，就根本没有什么直接确定的东西，所以我在就是一个非常确定的主观命题，同时也是一种绝对成见，那么，我们可以简单推断出先验知识的一些基本特点，他有两个和普通知识的区别

  1. 先验知识必须找出我之外有物存在的根源
  2. 先验知识必须区分我存在和我之外有物这两个命题，并尝试解决同一性

其次，先验哲学家认为主观的东西才具有原始实在性，让主观的对象直接成为自己的对象，客观的对象间接的成为他的对象。所以先验知识就是一种关于知识的知识，先验知识并不跳过主观直接对某种客观对象进行把握。

那么如何进行这种先验哲学思考呢？谢林的方法我认为和胡塞尔是非常类似的，就是尝试去把握一种持续的自我直观活动，通过直观活动看出被直观的东西，可以被认为是一种思维的加倍。把自己的思维本身作为一种直观的对象。

简而言之，先验考察方式的本性只能是主观的东西把自己变成自己对象的一种持续不断的活动。

先验哲学的初步划分

先验哲学考察的是知识本身，探讨知识是如何可能是先验哲学的重要课题，所有知识都能还原成某种原始成见，先验哲学的划分，取决于这种哲学承认的原始信念，所以我们接下来简单划分一下这种原始信念

  • A 我之外有物，且我之表象与物统一 —— 这是理论哲学，探讨经验的可能性
  • B 我心中的表象产生是自由的，能够从思想世界过渡到现实世界 —— 这是实践哲学，探讨自由行动的可能性
  • C A 和 B 实际上就是争夺到底是表象与对象为准，还是对象以表象为准的矛盾 —— 这就是先验哲学，探讨这个矛盾
  • D 如何解决这个矛盾呢，我们发现 A 代表了某种自然的创造性，B 代表了某种我的创造性，如何能建立两种创造的同一性，这就要通过艺术哲学来解决，因为艺术创作就是一种无意识(为何创造)+有意识(如何创造)的活动

艺术哲肯定了创造自然的无意识活动与有意志的有意识的创造性活动是同一的

艺术的理想世界和客体的现实世界都是同一种活动的产物，有意识的活动何无意识的活动两者的汇合无意识地创造力现实世界，而有意识地创造了美感世界

所以 理论哲学 + 实践哲学 + 艺术哲学 = 先验哲学

先验哲学的官能

先验考察的唯一直接的对象是主观的东西，因此作这类哲学思考的唯一官能上内在智能；理解哲学需要两个条件，人必须进行一种持续的创造活动，同时对这类活动进行持续的反映，人必须是被直观者(创造者)，同时还是直观者；为了进行创造和直观的二重化，这一过程必须通过想象力的美感活动；

超脱凡俗现实只有两条路，诗和哲学，前者使我们身临理想世界，后者使现实世界完全从我们面前消逝

关于先验唯心论的原理

关于知识的一个最高原理的必要性和性质

  1. 我们的知识一般有实在性，因为知识必须经过主观与客观的一致，所以必须通过某种普遍的中介
  2. 我们的知识应有一个体系，体系的条件是自身之内具有自己存在的根据，知识如果是体系，必然存在一个知识的原理于知识本身之内
  3. 这种知识的原理(起中介作用)只有一个
  4. 这个原理间接的是每门科学的原理，但直接地就是关于一切知识的科学，或者先验哲学的原理。
  5. 在先验哲学家看来，自我意识不是存在，是一种知识
  6. 我们为了在知识之内确立知识，无需考虑自我意识(第一知识)是否存在，通过这种限定，自我意识变成了一切知识的绝对原理，因为一切知识都必须以自我意识作为出发点
  7. 我们确认了知识在它自身内有一个绝对原理，同时也是作为科学的先验哲学的绝对原理，这个绝对原理限定这种科学的内容，也限定这种科学的形式
  8. 但是解释形式本身就需要一种知识学，同时也是一种科学，那么解释这种科学就又需要一种知识学，那么如何打破这种循环呢？需要找到一个绝对的点，在这个点中同时存在了内容和形式。
  9. 知识学就必须是一种同时是知识的形式，本身也是一种知识的存在，在这里，形式和内容互为条件，互相诱发

比如 A=A 这种逻辑，A = A 这种是逻辑的形式，但同时本身也是一种逻辑的内容(命题)

原理本身的演绎

接下来我们需要演绎出这种最高原理，也就是证明这种最高原理的地位。

1) 首先，我们的知识可以分为有条件的知识和无条件的知识，并且所有有条件的知识都可以达到一种无条件的知识

2) 无条件的知识是一种怎么样的知识呢？其实就是一种只以主观的东西作为条件，而不以客观的东西为条件的知识，就比如 A = A 这样的同一律，其核心只是表达一种同一性，并不关心A到底是什么，只受你思维的制约。

3) 但是作为一种知识，他必须是主观和客观的会合，才能成为知识，而 A = A 这种无条件的知识，那么这种无条件的知识如果能成为知识，那么A只能是某种实在的，客观的东西。

4) 分析命题(同一命题)是一种无条件的知识，它本身只包含思维内部的东西。综合命题是一种有条件的知识，他是主观与客观的会合。也就是说如果我们希望找到一个本原性的知识，一个绝对真实的知识，那么他必然同时是一个分析命题和综合命题(一方面这种知识必须是无条件的，一方面作为知识他必须是主观客观的会合)

5) 我们需要进一步分析同一命题和综合命题之间的关系，所有命题中都是两个概念进行相互对照，同一命题是概念和概念的对照(同一律，A和自己相等，A是概念，相等也是概念)，而综合命题则是一种概念和一个不同于概念的对象的对照。所以如果我们需要寻找一个同时是同一的也是综合的命题，那么就等于寻找一个点，这个点上的客体及其概念/对象及其表象就是同一个东西。

6) 既然存在是客观的东西，表象是主观的东西，那么这个课题也就意味着发现一个点，在这个点上主客同一

7) 那么什么东西是主客同一的呢？他肯定是一种被表象的东西同时也是作表象的东西(生成表象)，是一种被直观的东西同时也是进行直观的东西，那么很显然，这种东西只可能存在于自我意识之中。

接下来谢林讨论了大量关于自我意识结构的理论，基本就是关于理智直观(类似胡塞尔的本质直观)，把自我当成对象的一种理智直观活动，自我意识就是拥有这种构造的东西，简单来说就是自我的双重性让自我成为一种主观的同时也是客观的东西。篇幅较长，并且与胡塞尔的现象学一部分内容差不多，我这里就不过多阐述了。

自我是一种同时创造它自身(作为对象的)的知识活动

自我不是别的，只是一种把自身变成自己的对象的创造活动

接着谢林给出了哲学的第一原理，也就是 自我 = 自我 ，其中的含义不过是进行直观的自我和被直观的自我的同一性，因为被直观的自我是一个客体，所以也是一个综合命题。

知识的科学不能相信所有预设的客观，不能从客观出发，恰恰是要怀疑所有客观东西的实在性，客观的本质就在非客观的东西里面，也就是自我意识。

纯粹意识在用各种方式加以规定和限定时就得出了经验意识，从而诞生了经验自我，这种意识也正是幼儿说“我”的时候所代表的那个东西。存粹自我意识是一种活动，这种活动处在一切时间之外，一切时间正是由它构成的；经验意识只是在时间和表象的连续序列中产生的意识。

有一个常见的讨论是自我到底是不是存在的？这其实犯了一个基本的错误，所谓存在，是意识让它存在，自我意识是一种最高层次的活动，而存在不过的这种活动的某种表象或者说结果罢了！所以，自我是存在活动本身！这种活动赋予一切事物存在，它不需要任何东西设立它的存在，它自我支撑，客观上表现为永恒的生成(这里的生成英文为 Becoming 应该来源于为赫拉克利克，意味一种对象的不断变化)，主观上表现为无限的创造。

这个自我的最高原理必须同时也是实践的和理论的，否则就不能够同时作为实践哲学和理论哲学的最高原理，所以下一章节其实就是讨论如何从这个最高的原理出发，演绎出他是如何推导出实践哲学和理论哲学。

先验唯心论的一般演绎

既然前面已经证明了自我的二重性，那么其实也就证明了自我的存在(我能直接的感知到自我的存在)，自我 = 自我这个命题等于我在这个命题，而我在也是一种最高的实在性，因为所有东西都围绕我而存在。

对知识学的观念的证明的方法之一就是利用我在这个命题，推演出全部知识学体系。

如果说知识学其本性并不是单纯理论或是单纯实践(可以简单的猜出)，而必须同时是两者，那么我们必须像知识学那样，创立理论哲学和实践哲学，证明两者的对立，通过这种解释，方才能进一步的解释知识学的基础，解释这种对立的体系本身。

同时，自我到底是如何设定整个知识体系的? 客观世界的是如何产生的? 这些问题的解答也必须从自我意识出发。客观世界不过是一种我们固有的自由的受限制状态，那么，我们需要推演出在同一性的主体中，如何必然的诞生出一种受限定的活动和不受限定的活动，这种必然性必须同时是实践的，也是理论的。

所以接下来我们必须普遍的证明『自我若不同时是不受限制的活动，本来就无论如何都不能依靠自己的概念而成为受限制的』，简单来说就是『自我必须是不限制的活动，才能成为受限制的』，这种必然性是想要创立实践哲学和理论哲学必须先做出来的。

我们利用之前已经推导出来的这样一个命题来进行 『通过自我意识的活动，自我使自己成为自己的对象』

  1. 自我是自我的对象，而不是外物的对象
  2. 自我变成了自我的对象，说明自我首先不是对象(请看对2的进一步推论)

对2.的进一步推论

  1. 不是对象，就是对象的对立物，那自我原本就是活动，对象就是某种活动的对象。对象化本身就意味着一种被限定的意味，所以自我是一种无限的活动
  2. 自我是无限的活动，也是一切实在性的根据。(如果根据在自我之外，那就说明自我的受限的)
  3. 自我要产生自我(就如同自我意识的产生，自我的二重性想要产生)，就必须把自己设立成受限的对象，如此，自我意识才能产生(一个存粹自由的创造活动，没有对象，是无法创造自我的)
  4. 自我必须把自我设定成一种与自我相对的东西，才能限定自己的创造活动，自我把自我设定为限定的，创造了限定活动本身，那么难道自我把自身给扬弃了吗(自我的变质)？我们必须解决这个矛盾
  5. 我们需要证明“仅仅就自我是受到限定的而言，自我本身才能是不受限定的；反之，仅仅就自我是不受限定的而言，自我本身才能是收到限定的”，才能彻底说明自我的这种限定活动。

我们分别证明这两个命题

A 仅仅就自我是受到限定的而言，自我本身才能是不受限定的

  1. 自我就是它所是的一切，它仅仅是为它自身而存在的，自我是无限的为它自身而存在的 (如过是无限的并不为自身存在，那就不能称之为意识，是一种机器)
  2. 自我是无限的为它自身而存在的，也就是说在自我是为了直观而存在的，但是在直观的情况下，自我又被成为一种有限的，只有让自我重新变成一种无限的生成。
  3. 但是生成，必须通过一种有限的活动，设定界限才能生成
  4. 自我不应该是生成，而且是无限的生成，自我在不断的生成，从而扩大限制的边界，一直扩大到无限，但是并没有彻底取消限制

所以正是因为这种限制条件，才让自我成为无限。

B 反之，仅仅就自我是不受限定的而言，自我本身才能是收到限定的

自我之所以能够被限定为一种限定状态，就必须先建立一种无限的概念，所以自我的无限制状态就自我的有限制状态的条件

从AB两个命题出发 进一步推论

  1. A 可以得出限制是一种实在的，否则自我就不是真的受到限定的
  2. B 可以得出限制是一种观念的，否则自我就无法设定自己，把自己直观为受到限定的
  3. 那么如何解决呢？答案就是实在的限制是现实的活动，而观念的限制是观念的活动，人的意识活动就是这两种活动建立而成。
  4. 两个活动互为前提
  5. 理论哲学应该说明限制的观念性，实践哲学说明限制的现实性。理论哲学=唯心论，实践哲学=实在论，结合起来，才是先验唯心论

以先验唯心论为原则的理论哲学体系

我们由以出发的自我意识，是唯一的绝对活动，为了了解这种活动的本质(发生学机制)，我们在理论哲学当中的头一件事就是去推演这一绝对活动，为了发掘这种绝对活动，我们必须把它分解成一个个的单个活动，这些单个活动将会是那一绝对综合的中介环节。

为了研究自我意识，我们必须把我们的意识也就是主体统一体进行切割，接下来会分为两部分。1.首先对自我意识中包含的绝对综合进行推演，2. 随后找出这一绝对综合的中间环节。

自我意识所包含的绝对综合的演绎

  1. 【限制必须是观念的又是现实的】，那么限制就必然通过一种同时是观念和现实的活动产生，也就是自我意识
  2. 在自我意识彼岸，自我就是单纯的客观性(当你感受到的时候，他就已经是客观的东西了)。而作限制的活动是无法变成对象的，对象化等同于被限制，只有受到限制才能达到意识。所以我是无法看到这种作限制的活动的。（我们看到了这两种基本的活动）
  3. 如果单独拿出来这种作限制的活动，或者被限制的活动，那么这种活动必然不是自我
  4. 所以自我必然是一种更高的，结合了这两种活动的第三种活动产生的，自我也是一种复合的，综合的活动
  5. 这两种对立活动的斗争的本质就是一个不同方向的斗争，一方面自我具有创造性，所以向外延展，一方面需要自我直观，所以向内回归。
  6. 而这种对立活动之所以没有产生某一方的胜利，而是一直维持这种矛盾对抗状态，必然是因为有一种力量在维持这种对抗，维持自我的同一性
  7. 这种同一性必须通过一个中介才能存在，这种中介必然是摇摆于两种活动之间的一种活动
  8. 由于这种对抗是无限的，那么这种统一必须在一种无限的行动序列中才能得到统一，表现到自我意识的一个行动中。则是一个包含无限多的行动，也就是绝对综合的行动。

那么接下来，就是要讲述这种绝对综合的行动到底是如何运作的

首先，自我意识是绝对活动，通过这一活动，为自我确立了一切。这种原始的绝对活动，到底是怎样的一种活动呢，既然这种活动是构造意识的，在时间之外的活动，又该如何证明他的存在呢？

这种绝对活动的实在性只能通过推理得出，如果没有这种活动，则不可能由我的存在。其次，这种活动其实和哲学思考的活动本身是同一的活动，可以把哲学思考看成是这种活动的模仿。因为哲学思考的本质就是和表象拉开距离，通过哲学思考，我们可以打破表象的序列，从这种序列中跳出来。

哲学活动就好像是从前一个序列中跳出来，并进入了一个全新的序列，在前一个序列中必然的东西，在这一个序列中就变得是自由了。而哲学不过是开启一个新的序列之后，对那种原始序列的模仿(上面说的绝对活动)和复制(如果复制的完美，就是完美的哲学)。

对于第二个序列来说，第一个序列就是现实的，而自身是观念的。

PS: 这一部分我觉得其实胡塞尔对于内时间意识的讨论是比这里要更加好的。其实真正意义上的序列只有一个，也就是原始的绝对活动，而所有看起来实在的第一个序列，不过是一种意识的滞留

自我意识同时也是绝对对立活动的一种斗争，也就是无限进展的，被限制的活动VS观念的，不被限制的活动。我们必须理解观念性的活动如何得到限制，同时也必须讨论这种对抗活动的绝对综合到底是包含了哪些活动。

因此，哲学是自我意识的一部历史，这部历史由不同的时期，那个唯一的绝对综合就是由这些时期相继组合而成的

第一个时期 - 从原始感觉到创造性直观
